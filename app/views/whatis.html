What is this format?

This page describes the different formats of data that you might find on data.gov.uk.

CSV

CSV (Comma Separated Values) is a simple text based format that separates all of the values by a comma, or sometimes by a tab character.

Tools

CSV files can be opened in programs such as Excel, or LibreOffice. It is also a popular format with software developers as it is extremely easy to work with.

Go back

JSON

JSON (JavaScript Object Notation) is a format that is extremely popular with software developers, but isn't widely used outside of that area.

Tools

The most likely tools to work with JSON data are software development languages.

Go back

SHP

SHP files are often called Shapefile. They contain polygons and metadata for data that is intended for mapping purposes.

Tools

SHP files can be used with GIS tools such as QGIS, or ArcGIS.

Go back

WMS

WMS urls are Web Map Services that describe different layers of a map that are available. It is possible to overlay multiple layers on a map to see how things are related.

Tools

WMS urls can be used with GIS tools such as QGIS, or ArcGIS.
